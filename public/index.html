Virtual machine untuk workload apa pun

Buat dan jalankan VM online dengan mudah di infrastruktur cloud berperforma tinggi dan andal. Pilih jenis mesin kustom atau preset untuk server web, database, AI, dan lainnya.

Dapatkan satu instance VM e2-micro gratis per bulan. Pelanggan baru juga mendapatkan kredit gratis senilai $300 untuk mencoba Compute Engine dan produk Google Cloud lainnya.

Highlight produk

  • ✔️ Cara memilih dan men-deploy VM pertama Anda
  • ✔️ Mengenkripsi data sensitif dengan Confidential VMs
  • ✔️ Sesuaikan VM Anda untuk berbagai persyaratan komputasi, memori, performa, dan biaya

Anda dapat menghentikan pencarian dan mulai membuat keputusan yang tepat untuk memilih VM yang benar dengan Compute Engine.

Fitur

Konfigurasi kustom dan preset

Deploy aplikasi dalam hitungan menit dengan sampel bawaan yang disebut Solusi Praktis. Buat situs dinamis, VM dengan load balancing, aplikasi web tiga tingkat, atau aplikasi web e-commerce.

Pilih dari jenis, ukuran, dan konfigurasi mesin yang telah ditetapkan untuk workload apa pun, dari aplikasi perusahaan besar hingga workload modern (seperti container) atau project AI/ML yang memerlukan GPU dan TPU.

Untuk fleksibilitas lebih besar, buat jenis mesin kustom antara 1 hingga 96 vCPU dengan memori hingga 8,0 GB per core. Dan manfaatkan salah satu dari banyak opsi block storage, mulai dari Persistent Disk yang fleksibel hingga SSD Lokal berperforma tinggi dan berlatensi rendah.
